# VPython으로 키우는 사고력 — 파이썬 기초 30시간

> **"생각을 코드로, 코드를 3차원으로"**

컴퓨팅 사고력을 키우는 것이 목표입니다. VPython 3D 시각화를 도구로 삼아, 생각을 코드로, 코드를 3차원으로 구현합니다.

## 대상

프로그래밍 경험이 없는 중·고등학생 (주 대상: 고1), 30시간(15회 × 2시간) 수업

## 3방향 사고 훈련

매 챕터마다 세 가지 사고력 훈련을 합니다.

- **코드 → 결과 예측**: 코드를 읽고 실행 결과를 머릿속으로 그려보기
- **결과 → 코드 역추적**: 완성된 3D 결과를 보고 코드를 역으로 만들기
- **상상 → 코드 창작**: 머릿속 아이디어를 코드로 자유롭게 표현하기
