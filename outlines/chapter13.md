# 함수 — 나만의 명령어 만들기

**Part 5**: 함수와 종합 프로젝트

---

## 학습 목표

- [ ] def 키워드로 함수를 정의하고 호출할 수 있다
- [ ] 매개변수와 반환값의 개념을 이해하고 활용할 수 있다
- [ ] 반복되는 코드를 함수로 묶어 프로그램을 구조화할 수 있다

## 개요

함수를 '레시피' 또는 '자판기'에 비유한다. 물체 생성 함수, 거리 계산 함수 등을 직접 만들어본다. 매개변수로 다양한 물체를 생성하는 '공장 함수' 패턴을 배운다. '랜덤 도시 생성기' 프로젝트로 함수의 재사용성을 체감한다.
