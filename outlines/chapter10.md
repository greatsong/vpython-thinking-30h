# for 반복문 — 100개의 별을 한 번에

**Part 4**: 반복과 패턴의 마법

---

## 학습 목표

- [ ] for 반복문으로 같은 작업을 여러 번 수행할 수 있다
- [ ] range() 함수를 활용하여 반복 횟수를 제어할 수 있다
- [ ] 반복문으로 규칙적인 패턴(격자, 원형, 나선)을 생성할 수 있다

## 개요

반복문을 '복사기' 비유로 설명한다. for와 range로 물체를 한 줄로, 격자로, 원형으로 배치하는 실습을 한다. 수학적 패턴(피보나치 나선, 프랙탈 나무 등)을 3D로 시각화하여 반복의 힘을 체감한다.
