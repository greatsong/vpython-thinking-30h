# 변수 — 이름표를 붙여 기억하기

**Part 2**: 변수와 움직임의 세계

---

## 학습 목표

- [ ] 변수에 값을 저장하고 활용할 수 있다
- [ ] 물체를 변수에 담아 속성을 나중에 변경할 수 있다
- [ ] 변수 이름 짓기 규칙을 이해하고 의미 있는 이름을 사용할 수 있다

## 개요

변수를 '이름표가 붙은 상자' 비유로 설명한다. 숫자, 문자열, VPython 물체를 변수에 저장하는 실습을 한다. 변수를 통해 물체 속성(위치, 색상, 크기)을 동적으로 변경하는 방법을 배운다. '신호등 만들기' 프로젝트로 변수 활용을 실습한다.
