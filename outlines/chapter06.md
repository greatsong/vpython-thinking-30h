# 움직여라! — 애니메이션의 원리

**Part 2**: 변수와 움직임의 세계

---

## 학습 목표

- [ ] rate() 함수와 while True 루프로 애니메이션을 만들 수 있다
- [ ] 위치를 조금씩 변경하여 부드러운 움직임을 구현할 수 있다
- [ ] 속도(velocity) 개념을 코드로 표현할 수 있다

## 개요

애니메이션 = '그림을 빠르게 바꾸기'라는 원리를 설명한다. rate() 함수로 프레임 속도를 제어하고, 물체의 위치를 매 프레임 조금씩 변경하는 패턴을 배운다. 속도(velocity) 벡터 개념을 도입하여 공 던지기, 좌우 왕복 운동을 구현한다.
