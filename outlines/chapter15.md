# 최종 프로젝트 — 나만의 3D 시뮬레이션 만들기

**Part 5**: 함수, 위젯, 그리고 종합 프로젝트

---

## 학습 목표

- [ ] 지금까지 배운 모든 개념(변수, 조건, 반복, 함수, 위젯)을 종합하여 자유 프로젝트를 기획할 수 있다
- [ ] 뉴턴의 운동 법칙 등 물리 개념을 코드로 구현하여 시뮬레이션을 만들 수 있다
- [ ] 프로젝트를 단계별로 나누어 개발하고, 완성된 작품을 발표할 수 있다

## 개요

5가지 프로젝트 아이디어(태양계 시뮬레이션, 3D 게임, 물리 실험실 대시보드, 분자 구조 시각화, 건축 시뮬레이터)를 예시 코드와 함께 제시한다. 물리 시뮬레이션(F=ma, 중력, 탄성)의 기본 패턴을 가이드로 제공한다. 기획 → 핵심 기능 → 위젯 추가 → 발표의 4단계 개발 프로세스를 안내한다. 학생이 자유롭게 주제를 정하고 완성하며, 동료 피드백을 주고받는다.
