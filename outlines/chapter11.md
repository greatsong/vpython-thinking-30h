# while 반복문 — 끝나지 않는 시뮬레이션

**Part 4**: 반복과 패턴의 마법

---

## 학습 목표

- [ ] while 반복문과 for 반복문의 차이를 설명할 수 있다
- [ ] 종료 조건을 설정하여 안전한 반복을 구현할 수 있다
- [ ] dt(시간 간격)를 사용한 물리 시뮬레이션의 기본 원리를 이해한다

## 개요

while 반복을 '끝날 때까지 계속'이라는 비유로 설명한다. 무한 루프와 종료 조건, rate()를 결합한 시뮬레이션 패턴을 익힌다. 중력 가속도를 적용한 자유낙하 시뮬레이션을 만들고, dt(시간 간격) 개념으로 현실적인 물리 움직임을 구현한다.
