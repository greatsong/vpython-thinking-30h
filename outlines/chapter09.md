# 충돌 감지 — 부딪히면 어떻게 될까?

**Part 3**: 조건과 선택의 힘

---

## 학습 목표

- [ ] 두 물체 사이의 거리를 계산할 수 있다
- [ ] mag() 함수로 벡터의 크기를 구할 수 있다
- [ ] 충돌 조건을 설정하고 충돌 시 반응을 프로그래밍할 수 있다

## 개요

두 점 사이의 거리 개념을 3D 공간으로 확장한다. mag() 함수와 벡터 뺄셈으로 거리를 계산하고, 충돌 판정 로직을 구현한다. 공끼리 부딪히면 색이 바뀌거나 튕기는 시뮬레이션을 만든다. 간단한 '벽돌깨기' 게임으로 종합 실습한다.
