# 연산과 수식 — 물체를 계산으로 배치하기

**Part 2**: 변수와 움직임의 세계

---

## 학습 목표

- [ ] 사칙연산과 나머지 연산을 파이썬으로 수행할 수 있다
- [ ] 수식을 사용하여 물체의 위치와 크기를 계산할 수 있다
- [ ] 입력(input)을 받아 계산 결과를 3D로 보여줄 수 있다

## 개요

파이썬의 산술 연산자를 소개하고, 연산 결과를 3D 물체 배치에 활용한다. 원형으로 물체 배치하기(삼각함수 맛보기), 크기 비례 시각화 등을 실습한다. 사용자 입력을 받아 막대그래프를 3D로 그리는 프로젝트를 완성한다.

## 예상 소요 시간

2시간
