# 기여 가이드

이 교재는 오픈소스 프로젝트로, 선생님들의 기여를 환영합니다!

## 기여 방법

### 1. 저장소 Fork
이 저장소 우측 상단의 **Fork** 버튼을 클릭하여 본인 계정에 복사합니다.

### 2. 수정하기

**방법 A: GitHub 웹에서 직접 수정**
1. Fork한 저장소에서 수정할 파일(`docs/chapterXX.md`)을 클릭
2. 연필 아이콘(Edit) 클릭
3. 내용 수정 후 "Commit changes" 클릭

**방법 B: Claude Code 사용 (권장)**
```bash
git clone https://github.com/YOUR_USERNAME/vpython-thinking-30h.git
cd vpython-thinking-30h
# Claude Code로 수정
```

### 3. Pull Request 제출
1. Fork한 저장소에서 "Contribute" → "Open pull request" 클릭
2. 수정 내용을 설명하고 제출

## 기여 유형

| 유형 | 설명 | 난이도 |
|------|------|--------|
| 오타 수정 | 맞춤법, 문법 오류 교정 | 쉬움 |
| 내용 보강 | 설명 추가, 예시 보충 | 보통 |
| 새 챕터 | 새로운 주제의 챕터 작성 | 어려움 |
| 실습 자료 | 코드 예제, 연습문제 추가 | 보통 |

## 작성 규칙

- **마크다운** 형식으로 작성합니다
- 존댓말(합쇼체)을 사용합니다
- 코드 블록에는 언어를 명시합니다 (```python 등)
- 이미지는 `docs/images/` 폴더에 저장합니다
- 한 챕터는 A4 5~15페이지 분량을 권장합니다
