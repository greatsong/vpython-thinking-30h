# VPython으로 키우는 사고력 — 파이썬 기초 30시간

생각을 코드로, 코드를 3차원으로. GlowScript VPython을 도구로 삼아 '코드→결과 예측', '결과→코드 역추적', '상상→코드 창작'의 사고 훈련을 합니다.

**대상**: 프로그래밍 경험이 없는 중·고등학생 (주 대상: 고1), 30시간(15회 × 2시간) 수업

---

## Part 1: 3D 첫 걸음

GlowScript에서 3D 물체를 만들고, 좌표로 배치하고, 속성으로 꾸민다.

### Chapter 1: VPython의 세계로!

**학습 목표:**
- GlowScript에서 VPython 코드를 실행할 수 있다
- sphere()로 3D 물체를 만들 수 있다
- 코드 한 줄을 바꾸면 결과가 어떻게 달라지는지 예측할 수 있다

**개요:** GlowScript를 열고 sphere() 한 줄로 3D 공을 만든다. color를 바꿔보며 코드→결과 예측 훈련을 시작한다. 다양한 색과 크기의 공을 만들어보며 '한 줄이 3D 세계를 만든다'는 경험을 한다.

**예상 시간:** 2시간

---

### Chapter 2: 좌표의 비밀 — 3D 공간 이해하기

**학습 목표:**
- vector(x, y, z)로 물체를 원하는 위치에 배치할 수 있다
- 좌표를 보고 3D 배치를 종이에 예측할 수 있다
- x, y, z 축의 방향을 이해하고 활용할 수 있다

**개요:** 구슬 3개가 각각 다른 위치에 놓인 장면을 보고, vector의 숫자가 위치를 결정한다는 것을 발견한다. vector(2,3,0)은 어디일까? 종이에 예측하고 실행해서 확인하는 공간추론 훈련.

**예상 시간:** 2시간

---

### Chapter 3: 다양한 물체, 다양한 속성

**학습 목표:**
- box, cylinder, cone 등 다양한 3D 물체를 만들 수 있다
- 물체마다 다른 속성(radius, size, length 등)을 발견하고 활용할 수 있다
- 여러 물체를 조합하여 장면을 구성할 수 있다

**개요:** 상자, 원기둥, 원뿔이 다른 크기와 색으로 배치된 장면을 본다. sphere에는 radius, box에는 size가 있다는 패턴을 발견한다. 새 물체를 보면 '이건 어떤 속성이 있을까?' 추측하는 습관을 기른다.

**예상 시간:** 2시간

## Part 2: 변화와 움직임

변수로 물체를 기억하고, 랜덤으로 다양성을 만들고, 애니메이션으로 움직인다.

### Chapter 4: 변수와 계산 — 기억하고 조종하기

**학습 목표:**
- 변수에 물체를 저장하고 나중에 속성을 바꿀 수 있다
- 연산으로 물체의 위치를 계산할 수 있다
- '이름이 없으면 조종할 수 없다'는 변수의 필요성을 이해한다

**개요:** 공에 이름을 붙이지 않으면 나중에 색을 바꿀 수 없다는 문제를 직접 겪는다. 변수로 이름을 붙이면 ball.color로 조종 가능. 두 물체의 가운데 좌표를 계산해서 세 번째 물체를 놓는 실습.

**예상 시간:** 2시간

---

### Chapter 5: 랜덤의 마법 — 매번 다른 세계

**학습 목표:**
- random()으로 무작위 값을 생성할 수 있다
- 랜덤 위치, 랜덤 색, 랜덤 크기를 활용할 수 있다
- '같은 코드인데 매번 다른 결과'라는 확률적 사고를 경험한다

**개요:** 같은 코드를 실행할 때마다 다른 위치에 공이 나타난다. random()의 범위를 조절하면 결과가 어떻게 달라지는지 예측. 랜덤 색 + 랜덤 위치로 밤하늘의 별을 만드는 실습.

**예상 시간:** 2시간

---

### Chapter 6: 움직여라! — 애니메이션의 원리

**학습 목표:**
- while True + rate()로 애니메이션 루프를 만들 수 있다
- 위치를 매 프레임 변경하여 움직임을 구현할 수 있다
- 속도(velocity) 개념을 코드로 표현할 수 있다

**개요:** 공이 오른쪽으로 굴러간다. 매 프레임 pos.x += 0.1이 움직임의 비밀. 속도를 0.1에서 0.5로 바꾸면? 방향을 y로 바꾸면? 예측하고 실험하며 애니메이션 원리를 발견한다.

**예상 시간:** 2시간

## Part 3: 게임을 향해

조건문으로 판단하고, 키보드로 조종하고, 충돌을 감지하여 미니 게임을 만든다.

### Chapter 7: 벽에 부딪히면? — if문의 탄생

**학습 목표:**
- if문으로 조건에 따라 다른 동작을 실행할 수 있다
- 비교 연산자(>, <)를 활용할 수 있다
- '언제 방향을 바꿔야 하나?'라는 문제에서 조건문의 필요성을 발견한다

**개요:** 공이 화면 밖으로 사라진다. '벽에 닿으면 방향을 바꿔야 하는데...' → if문의 필요성. 벽 1면 → 4면 → 영역마다 색 다르게 점진적 확장.

**예상 시간:** 2시간

---

### Chapter 8: 키보드로 조종하기

**학습 목표:**
- keysdown()으로 키보드 입력을 감지할 수 있다
- 키 입력에 따라 물체를 이동시킬 수 있다
- 입력→동작 매핑을 스스로 설계할 수 있다

**개요:** 화살표 키로 공을 움직인다. 오른쪽 키→x 증가, 위 키→y 증가. 대각선 이동은? 두 키 동시 처리. 한 방향 → 4방향 → 아이템 먹기 점진 확장.

**예상 시간:** 2시간

---

### Chapter 9: 충돌 감지 — 부딪히면 어떻게 될까?

**학습 목표:**
- mag()로 두 물체 사이 거리를 계산할 수 있다
- 거리 < 반지름 합으로 충돌을 판단할 수 있다
- '만났다'를 수학적으로 표현하는 사고를 경험한다

**개요:** 두 공이 부딪히면 색이 바뀐다. '만났다'를 어떻게 코드로? → 거리 < 반지름 합. 고정 아이템 → 움직이는 아이템 → 여러 아이템 점진 확장.

**예상 시간:** 2시간

### Chapter 10: 미니 프로젝트 — 나만의 게임 만들기

**학습 목표:**
- Ch.4-9에서 배운 개념(변수, 랜덤, 애니메이션, if, 입력, 충돌)을 종합할 수 있다
- 게임 규칙을 설계하고 코드로 구현할 수 있다
- 단계별로 기능을 추가하며 프로젝트를 완성할 수 있다

**개요:** 지금까지 배운 모든 것을 조합한 미니 게임. 3가지 게임 아이디어 중 선택하거나 자유 기획. 단계별 가이드로 설계→분해→구현. 절반 지점 성취감.

**예상 시간:** 2시간

## Part 4: 패턴과 구조

for 반복문으로 대량 패턴을 만들고, 리스트로 물체를 그룹 관리한다.

### Chapter 11: for 반복문 — 패턴의 힘

**학습 목표:**
- for + range()로 반복 작업을 수행할 수 있다
- 1D → 2D → 3D 패턴을 만들 수 있다
- '100개를 3줄로' — 반복 패턴을 발견하고 코드로 표현할 수 있다

**개요:** 구슬 100개를 하나씩 만들면 100줄. 패턴을 발견하면 3줄. 한 줄 배치(1D) → 격자(2D, 5×5) → 큐브(3D, 3×3×3) 점진 확장. 축이 늘어도 코드 변화는 최소.

**예상 시간:** 2시간

---

### Chapter 12: 리스트 — 물체를 모아서 관리하기

**학습 목표:**
- 리스트에 여러 물체를 저장하고 접근할 수 있다
- for + 리스트로 모든 물체를 동시에 제어할 수 있다
- '이름 10개 vs 리스트 1개' — 그룹 관리의 필요성을 체감한다

**개요:** 별 10개가 각각 다른 속도로 떨어진다. star1, star2... 로 관리하면 불가능 → 리스트. 만들고 접근 → for+리스트 동시 이동 → 공끼리 충돌 점진 확장.

**예상 시간:** 2시간

### Chapter 13: 함수 — 나만의 명령어 만들기
