## 📘 프로젝트 개요

### 책 제목
**VPython으로 키우는 사고력 — 파이썬 기초 30시간**

### 핵심 철학
**"생각을 코드로, 코드를 3차원으로"**

- **컴퓨팅 사고력(Computational Thinking)**이 이 교재의 궁극적 목표다
- VPython은 사고를 3차원으로 가시화하는 도구 — 코딩 자체가 목적이 아님
- IQ 테스트처럼 사고력을 자극하는 3가지 핵심 훈련:
  1. **결과 → 코드** (역추적): 3D 결과물을 보고 그것을 만드는 코드를 작성한다
  2. **코드 → 결과** (예측): 코드를 읽고 실행 결과가 어떨지 머릿속에서 그려본다
  3. **상상 → 코드** (창작): 머릿속 아이디어를 코드로 구현하여 3D로 실현한다
- 매 챕터마다 이 3가지 사고 훈련을 반복하여 컴퓨팅 사고의 근육을 키운다
- 패턴 인식, 추상화, 분해, 알고리즘 설계 — CT의 4대 요소를 VPython으로 체득

### 대상 독자
- 프로그래밍 경험이 전혀 없는 중학생~고등학생 (주 대상: 고1)
- 수학적 사고에 약간의 관심이 있거나, 게임/시뮬레이션에 흥미가 있는 학생
- 30시간 수업 (15회 × 2시간) — 방과후학교, 자유학기, 동아리 등
- 개인 노트북 또는 컴퓨터실 환경 (인터넷 접속 가능)

### 자료 성격
**컴퓨팅 사고력 훈련을 위한 VPython 3D 시각화 기반 파이썬 입문 교재**
- "코드 → 결과 예측", "결과 → 코드 역추적", "상상 → 코드 창작"의 3방향 사고 훈련이 모든 챕터를 관통
- 전통적 프로그래밍 개념(변수, 조건, 반복, 함수)을 가르치되, 매번 3D 시각화와 사고력 문제로 포장
- 매 챕터에 "🧠 사고력 챌린지" 코너를 두어 IQ 테스트 스타일의 문제를 출제
- 점진적 난이도 상승: 단순 물체 생성 → 패턴 만들기 → 움직임 예측 → 시뮬레이션 설계

## ✍️ 작성 원칙

### 설명 원칙
**3단계 설명법**:
1. 비유로 시작 - "벡터는 마치 화살표와 같아요. 방향과 크기를 동시에 알려줍니다"
2. 정확한 정의 - "정확히 말하면, VPython의 vector(x, y, z)는 3차원 공간에서의 위치나 방향을 나타냅니다"
3. 예시로 확인 - "예를 들어, vector(1, 0, 0)은 오른쪽을 가리키는 화살표입니다"

### 코드 원칙
```python
# === WHAT: 빨간 공을 만들고 오른쪽으로 움직인다 ===
# --- WHY: 위치 변화가 곧 "운동"이라는 개념을 이해하기 위해 ---
ball = sphere(pos=vector(0,0,0), radius=0.5, color=color.red)  # HOW: 원점에 반지름 0.5인 빨간 공 생성
ball.pos.x += 1  # HOW: x좌표를 1 증가시켜 오른쪽으로 이동
```

**복사-실행 원칙**: 모든 코드는 복사해서 바로 실행 가능해야 함
**VPython 특화 원칙**:
- `from vpython import *`는 모든 코드 상단에 포함
- GlowScript 환경과 로컬 환경 모두 호환되도록 작성
- 3D 장면의 예상 모습을 텍스트로 설명 ("실행하면 빨간 공이 화면 중앙에 나타납니다")

### 문법과 사고력의 균형 원칙 (⭐ 핵심)
매 챕터에서 파이썬 문법을 정확히 가르치면서 동시에 사고력을 훈련한다:
- **문법 박스**: 새로 배우는 파이썬 문법을 정식 명칭과 함께 정리 (예: "📖 파이썬 문법: while 반복문")
- **사고력 챌린지**: 문법을 활용한 3종류의 사고력 문제를 반드시 포함
- 문법을 "외우는 것"이 아니라 "이해하고 활용하는 것"이 되도록 설계

### 시각화 원칙
- Mermaid 다이어그램 대신 **VPython 미니 코드**로 개념을 시각화한다
- VPython 코드 자체가 곧 그림이므로, 개념 설명도 3D 물체로 보여준다
- 실행 결과가 궁금한 부분은 텍스트로 상세히 묘사한다 ("실행하면 ~가 보입니다")

### 사고력 훈련 원칙
매 챕터에 3종류의 사고력 문제를 반드시 포함:

**🔍 코드 → 결과 예측**: 코드를 보여주고 "실행하면 어떤 모습일까?" 질문
- 머릿속에서 코드를 실행하는 "멘탈 시뮬레이션" 능력 훈련
- 정답은 접기(details)로 숨기고, 직접 예측 후 확인

**🔄 결과 → 코드 역추적**: 3D 결과물(텍스트 설명)을 보여주고 "이걸 만드는 코드를 작성하라"
- 분해(decomposition) + 알고리즘 설계 능력 훈련
- 단서를 하나씩 제공하는 힌트 시스템

**💡 상상 → 코드 창작**: 주제만 주고 자유롭게 구현
- 추상화 + 창의적 문제 해결 능력 훈련
- "정답 없음"을 명시, 다양한 접근을 격려

## 📝 메모

- VPython은 `pip install vpython`으로 설치하거나, GlowScript(glowscript.org)에서 웹으로 실행 가능
- 설치 문제를 최소화하기 위해 GlowScript 사용을 기본으로 안내하되, 로컬 설치 방법도 부록으로 제공
- 챕터 후반부로 갈수록 물리 개념(중력, 탄성, 파동 등)이 등장하지만, 수식보다 시뮬레이션 결과로 직관적 이해를 우선
- 최종 프로젝트는 학생 스스로 주제를 정하고 3D 시뮬레이션을 만드는 자유 프로젝트
- 매 챕터 시작에 "이번 시간에 만들 것" 미리보기(텍스트 설명) 제시하여 흥미 유발
- 🧠 사고력 챌린지는 난이도를 ⭐~⭐⭐⭐으로 표시하여 도전 욕구 자극
