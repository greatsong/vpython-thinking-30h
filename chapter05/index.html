    Ch.5 — 랜덤의 마법¶

    Part 2: 변화와 움직임 | 핵심: random(), 범위 조절

    🎬 오늘의 장면¶

    밤하늘을 올려다본 적 있나요?

    별은 누가 자리를 정해준 게 아닙니다. 제각각 다른 위치에, 다른 밝기로 빛나고 있죠.

    오늘 우리는 컴퓨터로 그런 밤하늘을 만들어 봅니다. 그런데 신기한 일이 벌어집니다. 같은 코드를 실행할 때마다 별의 위치와 색이 달라져요. 한 글자도 바꾸지 않았는데!

    비밀은 바로 random()이라는 함수에 있습니다.

    밤하늘과 꽃밭 — 실행할 때마다 달라져요!

    🔍 코드 읽기 챌린지¶

    아래 코드를 천천히 읽어 보세요. 아직 실행하지 마세요!

    GlowScript 3.2 VPython

scene.background = vector(0.05, 0.05, 0.15)

sphere(pos=vector(random()*4, random()*3, 0), radius=0.2, color=color.yellow)
sphere(pos=vector(random()*4, random()*3, 0), radius=0.2, color=color.yellow)
sphere(pos=vector(random()*4, random()*3, 0), radius=0.2, color=color.yellow)
    질문 1: random()은 어떤 값을 돌려줄까요?

    정확한 숫자는 아무도 모릅니다! 그게 바로 랜덤이니까요. 하지만 범위는 알 수 있어요. random()은 0 이상 1 미만의 실수를 돌려줍니다. 예를 들어 0.7382… 같은 값이요.

    질문 2: random()*4는 어떤 범위일까요?

    random()이 0~1 사이니까, 거기에 4를 곱하면 0~4 사이가 됩니다. 0.0일 수도 있고, 3.14일 수도 있고, 1.87일 수도 있어요.

    질문 3: 공 3개의 위치가 서로 같을까요, 다를까요?

    거의 확실하게 다릅니다! 각 줄의 random()은 매번 새로운 무작위 값을 만들어 냅니다. 그래서 세 공은 제각각 다른 곳에 나타나요.

    이제 실행해 보세요. 그리고 한 번 더 실행해 보세요. 위치가 달라졌죠?

    🛠️ 직접 만들어보기¶

    단계 1: random()을 만나다¶

    가장 단순한 형태부터 시작합니다.

    GlowScript 3.2 VPython

sphere(pos=vector(random(), 0, 0), radius=0.3, color=color.red)

    실행할 때마다 공의 x 위치가 달라집니다. 0과 1 사이 어딘가에 나타나요.

    여러 번 실행해 보세요. 공이 매번 다른 곳에 있죠?

    단계 2: 범위를 넓히기¶

    0~1은 너무 좁습니다. 곱하기로 범위를 넓혀 봅시다.

    GlowScript 3.2 VPython

sphere(pos=vector(random()*8, 0, 0), radius=0.3, color=color.red)

    이제 공이 x좌표 0~8 사이 어딘가에 나타납니다. 훨씬 넓어졌죠?

    범위 조절 공식

    random() * N → 0 ~ N 사이의 랜덤 값을 만듭니다. N이 클수록 범위가 넓어져요.

    단계 3: 음수 범위까지¶

    화면 가운데를 기준으로 왼쪽(-), 오른쪽(+) 모두 사용하고 싶다면?

    GlowScript 3.2 VPython

sphere(pos=vector(random()*6 - 3, 0, 0), radius=0.3, color=color.blue)

    random()*6은 0~6 사이 값을 만들고, 거기서 3을 빼면 -3 ~ 3 사이가 됩니다!

    중심 맞추기 공식

    random() * N - N/2 → -N/2 ~ N/2 사이. 가운데(0)를 중심으로 양쪽에 퍼집니다.

    단계 4: 랜덤 색상¶

    위치뿐 아니라 색도 랜덤으로 만들 수 있습니다.

    GlowScript 3.2 VPython

scene.background = vector(0.1, 0.1, 0.1)

sphere(pos=vector(0, 0, 0), radius=0.5,
       color=vector(random(), random(), random()))

    vector(R, G, B)에서 R, G, B가 모두 랜덤이니까 실행할 때마다 완전히 다른 색이 나타납니다. 분홍이 될 수도 있고, 연두가 될 수도 있고, 갈색이 될 수도 있어요.

    단계 5: 랜덤 크기¶

    크기도 랜덤으로 만들어 봅시다.

    GlowScript 3.2 VPython

sphere(pos=vector(-2, 0, 0), radius=random()*0.5 + 0.1,
       color=color.cyan)
sphere(pos=vector(0, 0, 0), radius=random()*0.5 + 0.1,
       color=color.magenta)
sphere(pos=vector(2, 0, 0), radius=random()*0.5 + 0.1,
       color=color.yellow)

    random()*0.5 + 0.1은 0.1 ~ 0.6 사이의 반지름을 만듭니다. 너무 작아서 안 보이는 걸 방지하려고 0.1을 더했어요.

    단계 6: 밤하늘 별 만들기¶

    이제 모든 것을 합쳐 봅시다!

    GlowScript 3.2 VPython

scene.background = vector(0.05, 0.05, 0.15)

sphere(pos=vector(random()*10-5, random()*6-3, 0),
       radius=0.1, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*6-3, 0),
       radius=0.1, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*6-3, 0),
       radius=0.1, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*6-3, 0),
       radius=0.1, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*6-3, 0),
       radius=0.1, color=vector(1, 1, random()))

    다섯 개의 별이 밤하늘에 흩뿌려집니다. 색은 vector(1, 1, random())이라서 노란빛~흰빛 사이의 별빛이 나와요. R=1, G=1, B=랜덤이니까요.

    같은 줄을 다섯 번이나 복사해야 한다니, 좀 번거롭죠? 나중에 Ch.11에서 for를 배우면 한 줄로 100개도 만들 수 있습니다. 지금은 조금만 참아 주세요!

    🔄 역추적 챌린지¶

    결과를 먼저 보고, 코드를 거꾸로 추론해 보세요.

    문제 1: 왼쪽 절반에만 공 만들기¶

    화면의 왼쪽 절반(x가 -5 ~ 0)에만 랜덤 공이 나타나게 하고 싶어요.

    힌트

    random()은 0~1 사이 값을 만듭니다. 여기에 뭘 곱하고 뭘 빼면 -5~0 사이가 될까요?

    정답

    random()*5 - 5를 사용하면 됩니다!

    • random()*5 → 0~5 사이
    • random()*5 - 5 → -5~0 사이
    GlowScript 3.2 VPython
sphere(pos=vector(random()*5-5, random()*4-2, 0),
       radius=0.3, color=color.green)

    문제 2: 따뜻한 색만 나오게¶

    빨강~노랑 계열의 따뜻한 색만 랜덤으로 나오게 하고 싶어요.

    힌트

    따뜻한 색은 R이 높고, G는 중간, B는 낮은 색입니다. 어떤 값을 고정하고 어떤 값을 랜덤으로 할까요?

    정답

    R은 1로 고정, G는 랜덤(0~1), B는 0으로 고정하면 빨강~노랑이 나옵니다!

    GlowScript 3.2 VPython
sphere(pos=vector(0, 0, 0), radius=0.5,
       color=vector(1, random(), 0))

    📖 알고 넘어가기¶

    random()의 비밀¶

    random() 함수는 호출할 때마다 0 이상 1 미만의 실수를 하나 돌려줍니다. 예를 들어 0.28173... 같은 값이에요.

    핵심은 이겁니다. 어떤 값이 나올지 아무도 모른다. 우리가 아는 건 범위뿐이에요.

    GlowScript VPython에서는 별도의 import 없이 그냥 random()이라고 쓰면 됩니다.

    범위 조절하기¶

    • random() → 0 ~ 1
    • random() * N → 0 ~ N
    • random() * N + A → A ~ A+N
    • random() * N - N/2 → -N/2 ~ N/2 (가운데 정렬)

    빠른 계산법

    범위를 만들고 싶으면 두 가지만 생각하세요. 1) 범위의 폭 → 곱하기 값. 2) 시작점 → 더하기(또는 빼기) 값.

    예: 2 ~ 8 사이를 원한다면? 폭 = 8 - 2 = 6, 시작점 = 2 → random()*6 + 2

    랜덤 색, 랜덤 위치, 랜덤 크기¶

    random()은 숫자가 들어가는 곳이면 어디든 쓸 수 있습니다.

    • 랜덤 위치: pos=vector(random()*10-5, random()*6-3, 0)
    • 랜덤 색: color=vector(random(), random(), random())
    • 랜덤 크기: radius=random()*0.5 + 0.1

    확률적 사고¶

    프로그래밍에서는 보통 "같은 코드 = 같은 결과"입니다. 하지만 random()이 들어가면 "같은 코드인데 매번 다른 결과"가 됩니다. 이것을 확률적(probabilistic) 사고라고 합니다.

    게임의 아이템 드롭, 날씨 시뮬레이션, AI의 창작 — 모두 이 random()에서 시작합니다.

    🐛 버그 사냥¶

    아래 코드에는 버그가 있습니다. 찾아서 고쳐 보세요!

    버그 1: 인자를 넣었더니¶

    GlowScript 3.2 VPython

sphere(pos=vector(random(10), 0, 0), radius=0.3, color=color.red)
    버그 정체

    GlowScript의 random()은 인자를 받지 않습니다! random(10)이라고 쓰면 오류가 나요.

    0~10 사이 값을 원하면 random()*10으로 써야 합니다.

    GlowScript 3.2 VPython
sphere(pos=vector(random()*10, 0, 0), radius=0.3,
       color=color.red)

    버그 2: -3~3 사이를 원했는데¶

    GlowScript 3.2 VPython

sphere(pos=vector(random()*6, 0, 0), radius=0.3, color=color.blue)
    버그 정체

    random()*6의 범위는 0~6입니다. -3~3이 아니에요!

    -3~3 사이를 만들려면 3을 빼야 합니다: random()*6 - 3

    GlowScript 3.2 VPython
sphere(pos=vector(random()*6 - 3, 0, 0), radius=0.3,
       color=color.blue)

    버그 3: 별이 안 보여요¶

    GlowScript 3.2 VPython

scene.background = vector(1, 1, 1)
sphere(pos=vector(0, 0, 0), radius=0.1,
       color=vector(1, 1, random()))
    버그 정체

    배경이 흰색(1,1,1)인데, 별 색이 vector(1, 1, random())이라서 거의 흰색에 가깝습니다. 흰 배경에 흰 별은 안 보이죠!

    밤하늘을 만들려면 배경을 어둡게 바꿔야 합니다.

    GlowScript 3.2 VPython
scene.background = vector(0.05, 0.05, 0.15)
sphere(pos=vector(0, 0, 0), radius=0.1,
       color=vector(1, 1, random()))

    💡 상상 챌린지¶

    random()으로 자연 풍경을 만들어 보세요!

    도전 1: 꽃밭¶

    초록 바닥 위에 랜덤 위치, 랜덤 색의 꽃(작은 구)을 여러 개 놓아 보세요.

    힌트

    바닥은 box()로 만들고, 꽃은 sphere()를 여러 개 복사하세요. 위치의 y좌표를 0 근처로 고정하면 바닥 위에 놓입니다.

    예시 코드 
    GlowScript 3.2 VPython
scene.background = vector(0.5, 0.8, 1)
box(pos=vector(0,-0.5,0), size=vector(10,0.2,6),
    color=vector(0.2, 0.6, 0.1))
sphere(pos=vector(random()*8-4, 0, random()*4-2),
       radius=0.15, color=vector(1, random(), random()*0.3))
sphere(pos=vector(random()*8-4, 0, random()*4-2),
       radius=0.15, color=vector(1, random(), random()*0.3))
sphere(pos=vector(random()*8-4, 0, random()*4-2),
       radius=0.15, color=vector(random(), random(), 1))
sphere(pos=vector(random()*8-4, 0, random()*4-2),
       radius=0.15, color=vector(1, random()*0.5, 0.8))
sphere(pos=vector(random()*8-4, 0, random()*4-2),
       radius=0.15, color=vector(1, 1, random()))

    도전 2: 나만의 랜덤 세계¶

    위치, 색, 크기, 도형 종류를 조합해서 여러분만의 장면을 만들어 보세요. sphere, box, cylinder, cone — Ch.3에서 배운 모든 도형을 쓸 수 있습니다!

    📝 오늘의 완성 코드¶

    별이 빛나는 밤하늘, 그 아래 랜덤 꽃밭이 있는 풍경입니다.

    GlowScript 3.2 VPython

# 📝 어두운 밤하늘 배경 설정
scene.background = vector(0.05, 0.05, 0.15)

# 📝 별 5개 — 위치와 밝기가 모두 랜덤
# 📝 random()*10-5 → x가 -5~5 사이에 흩뿌려짐
# 📝 random()*3+1 → y가 1~4 사이 (하늘 위쪽)
# 📝 vector(1,1,random()) → R=1, G=1, B=랜덤 → 노랑~흰색 별빛
sphere(pos=vector(random()*10-5, random()*3+1, 0),
       radius=0.08, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*3+1, 0),
       radius=0.1, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*3+1, 0),
       radius=0.06, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*3+1, 0),
       radius=0.12, color=vector(1, 1, random()))
sphere(pos=vector(random()*10-5, random()*3+1, 0),
       radius=0.07, color=vector(1, 1, random()))

# 📝 초록 땅 — box로 넓은 바닥을 깔아줌
box(pos=vector(0, -1.5, 0), size=vector(12, 0.3, 6),
    color=vector(0.15, 0.4, 0.1))

# 📝 꽃 5송이 — 랜덤 위치 + 따뜻한 랜덤 색 + 랜덤 크기
# 📝 random()*0.15+0.08 → 반지름 0.08~0.23 (너무 작지 않게)
sphere(pos=vector(random()*8-4, -1.2, random()*3-1.5),
       radius=random()*0.15+0.08,
       color=vector(1, random()*0.5, random()*0.3))
sphere(pos=vector(random()*8-4, -1.2, random()*3-1.5),
       radius=random()*0.15+0.08,
       color=vector(1, random()*0.3, random()*0.5))
sphere(pos=vector(random()*8-4, -1.2, random()*3-1.5),
       radius=random()*0.15+0.08,
       color=vector(random()*0.3, random()*0.5, 1))
sphere(pos=vector(random()*8-4, -1.2, random()*3-1.5),
       radius=random()*0.15+0.08,
       color=vector(1, 1, random()*0.3))
sphere(pos=vector(random()*8-4, -1.2, random()*3-1.5),
       radius=random()*0.15+0.08,
       color=vector(1, random(), random()*0.2))
    실행 결과 — 밤하늘과 꽃밭 (매번 달라져요!)

    실행할 때마다 별의 위치, 꽃의 색과 크기가 달라집니다. 한 글자도 바꾸지 않았는데 매번 새로운 세계가 펼쳐져요!

    ✅ 3줄 정리¶

    오늘 배운 것

    1. random()은 0~1 사이 무작위 실수를 만든다 — 호출할 때마다 다른 값!
    2. random()*N + A로 원하는 범위를 만든다 — 곱하기는 폭, 더하기는 시작점
    3. 위치, 색, 크기 어디든 random()을 넣으면 같은 코드, 매번 다른 결과!
